# Exemplo de Descrição de Vaga

## Desenvolvedor Python Sênior

### Sobre a Empresa

Somos uma empresa de tecnologia inovadora focada em transformar a experiência do usuário através de soluções inteligentes baseadas em IA. Nossa missão é democratizar o acesso à tecnologia de ponta e criar produtos que fazem a diferença na vida das pessoas.

### A Oportunidade

Estamos buscando um Desenvolvedor Python Sênior apaixonado por tecnologia e inovação para se juntar ao nosso time de engenharia. Você terá a oportunidade de trabalhar em projetos desafiadores que impactam milhares de usuários diariamente.

### Responsabilidades

- Desenvolver e manter sistemas backend escaláveis usando Python
- Colaborar com equipes multidisciplinares para definir e implementar novas funcionalidades
- Escrever código limpo, testável e bem documentado
- Participar de revisões de código e contribuir para a melhoria contínua do time
- Mentorear desenvolvedores júnior e compartilhar conhecimento

### Requisitos

**Obrigatórios:**
- 5+ anos de experiência com desenvolvimento Python
- Experiência sólida com frameworks web (Django, Flask ou FastAPI)
- Conhecimento profundo de bancos de dados relacionais e NoSQL
- Experiência com APIs RESTful e microsserviços
- Familiaridade com Git e práticas de CI/CD

**Desejáveis:**
- Experiência com cloud (AWS, GCP ou Azure)
- Conhecimento de Docker e Kubernetes
- Experiência com machine learning e IA
- Contribuições em projetos open source

### Benefícios

- Salário competitivo compatível com o mercado
- Vale refeição e vale transporte
- Plano de saúde e odontológico
- Trabalho remoto flexível
- Orçamento para cursos e conferências
- Ambiente colaborativo e inovador
- Oportunidades de crescimento profissional
